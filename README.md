## gdb常用命令
- run : 从头开始（mian函数）运行程序，直到遇到断点停下。
- continue：从【当前位置】开始连续运行程序，直到遇到断点停下。
- start：开始执行程序，【停在main函数第一行语句前面等待命令】。
- finish：【连续运行到当前函数返回为止】，然后停下来等待命令。
-【如果某个函数的局部变量发生访问越界，有可能并不立即产生段错误，而是在函数返回时产生错误。】
